# 设备模型

设备模型的目的是复用物模型和图表。如下说明设备模型功能如何使用。

## 1、说明
- 点击设备接入-设备模型进入模板管理页面。
- 设备模型对应了现实世界的产品名，例如XX品牌的温湿度传感器中的"温湿度传感器"
- 设备模型的目的是定义物模型定义图表D
- App端的图表等，设定预配置，方便高效的接入设备。
- 设备模型可以被设备模型调用。


## 2、操作
